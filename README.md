# deFeeter

#### A Java application that allows for quick conversion from feet (among other diabolical imperial measurements) to metres.
- Quickly convert feet and inches to meter
- Small and efficient; Perfect for all your imperial hate crimes!
- Too lazy to pull up a convertion website everytime? This apps the one for you!
- Purely Java (and Java Swing)

Please note that the app is incomplete; There may be changes to the current app and/or new features added in the near future.
However be rest assured that there will be no metric to imperial convertions; The point of the app is to spite imperial measurements :)

### How to use?
1. Press the desired convertion's button
2. Enter the measurement to be converted. **Please follow the format that is asked in the windows.** It may not convert properly. Regarding inches, it is expected that the inch value only goes from 0-12. This may be changed at a future date.
3. Press the button
4. ✨ **Huzzah!** ✨ The converted value should show up at the bottom of the page.
